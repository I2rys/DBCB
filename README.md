## Installation
Github:

    git clone https://github.com/I2rys/DBCB
    
NPM Packages:
```
npm i discord.js
```

## Setup
1. Open index.js
2. Find **Self** > **discordbot_token** and replace it with your Discord bot token.

## Usage
```
node index.js
```
